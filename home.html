Weapons Inventory Analytics System
Data-driven application for efficient weapon spares management

The WIAS application helps the user to get actionable insights from the weapon inventory held at various WEDs. It supports a number of features such as a detailed overview of inventory held at different WEDs, compliance of various depots for demands raised by units, consumption patterns of various equipment based on class of ships etc. The application also generates an inventory optimization model combining both ABC analysis as well as Fast-Slow-Non-moving item analysis. Additionally, WIAS deploys a state of the art machine learning model (Croston Model with xgboost for hyper-parameter tuning) for predicting the expected consumption of spares for a given item.
